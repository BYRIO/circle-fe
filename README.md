# MUGI.WIKI
The frontend (Vue) of MUGI Project

## Command

```bash
npm install
# webpack build
npm run build
# mock server localhost:3000
npm run mock
```

## Technology Stack

* Webpack
* JavaScript
    * ECMAScript 6
    * lodash
    * Babel
    * ESLint
* MVVM
    * Vue
    * Vuex
    * vue-router
    * axios
    * json-server // for mock
* CSS
    * PostCSS
    * cssnext
